Production-grade, not a prototype

Backend

  • Next.js 16 App Router + Turbopack
  • Prisma 7 + PostgreSQL (adapter-pg)
  • JWT auth with HTTP-only cookies
  • Idempotent webhook with HMAC guard
  • 42 unit tests + 6 E2E Playwright specs

Bolna V2 Integration

  • Agent: de639b14 (Priya, logistics specialist)
